# Tahja Quran Learning App (Prototype)

## Overview

This is a web-based prototype of a Quran learning application. It allows users to explore surahs and ayahs, and navigate through words step-by-step.

##  Features

* Display all 114 surahs using Quran API
* Search surah by name or number
* View ayahs of selected surah
* Word-by-word navigation using keyboard (ArrowLeft)
* Interactive UI with highlighting and selection

## Technologies Used

* HTML
* CSS
* JavaScript
* Quran API (alquran.cloud)

## Note

This is a prototype project. No real hardware or gesture input is implemented.
